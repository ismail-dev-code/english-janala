Frequently Asked Questions

1. What is the difference between var let const in js?
Var: Var accessible throughout the entire function where it's declared and we can reassign and redeclare var in the same scope. Its a function-scoped.
Const: We use const for values that will not change and its a block-scoped like let.
Let: We use let when we need to reassign a value and it’s accessible only within the block {} where it's declared. We can reassign let but cannot redeclare it in the same scope.
2. What is the difference between map() forEach() and filter() in js?
map(): map() creates a new array by applying a function to each element in the original array.
forEach(): forEach() executes a function on each element in the array but does not return a new array.
filter(): filter() creates a new array with only the elements that satisfy a given condition (returns true).

4. How do javascript promises work?
Promises are used to handle asynchronous operations — like fetching data from an API, reading a file, or doing any task that takes time to complete — without blocking the execution of other code.
